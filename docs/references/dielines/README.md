# Die-Line Referansları

Bu klasör, Sade Chocolate kutu tasarımları için die-line (kesim şablonu) referanslarını içerir.

---

## Dosyalar

### 1. `01-tuck-end-box.webp`
**Tip:** Tuck End Box / Sleeve Box
- Asma delikli (hang tab) versiyon
- Pencere kesimi (window cutout) opsiyonu
- **Kullanım:** Tablet çikolata, bar ambalaj

**Die-line özellikleri:**
- Düz çizgi = Kesim hattı (cut line)
- Turuncu kesikli = Katlama hattı (fold line)

---

### 2. `02-tray-lid-box.jpg`
**Tip:** Tray + Lid Box (Tepsi + Kapak)
- (a) Tepsi (tray): Alt kısım
- (b) Kapak (lid): Dekoratif pencere deseni ile

**Kullanım:** Premium çikolata kutuları, hediye setleri
**Sade Chocolate için ideal!**

### İkincil (Secondary)

## Die-Line Standartları

### Çizgi Renkleri (Sektör Standardı)
| Renk | Anlam |
|------|-------|
| Siyah düz | Kesim hattı (cut line) |
| Kırmızı/Turuncu kesikli | Katlama - Dağ (mountain fold) |
| Mavi kesikli | Katlama - Vadi (valley fold) |
| Yeşil | Yapıştırma alanı (glue tab) |
| Pembe | Pencere kesimi (window cut) |

### Ölçüler
- Birim: mm (milimetre)
- Bleed: 3mm (tüm kenarlarda)
- Safety zone: 3mm (kesimden içeride)
- Minimum malzeme kalınlığı: 250-350 gsm karton
